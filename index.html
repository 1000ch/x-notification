<x-notification>

Declarative Browser Notification as Web Components.

Star Follow @1000ch

Install

# via npm
$ npm install x-notification

Usage

Import x-notification.js and register.

<script type="module">
import XNotification from 'https://cdn.jsdelivr.net/npm/x-notification/dist/index.js';
customElements.define('x-notification', XNotification);
</script>

After loading, put <x-notification> tag.

<x-notification autoshow title="Notification Title" delay="1000" timeout="3000" tag="tag">
  Notification body
</x-notification>

Autoshow Example

<x-notification autoshow title="Hello!" delay="0" timeout="3000">
  This will be shown automatically.
  Delay is 0, timeout is 3000.
</x-notification>
This will be shown automatically. Delay is 0, timeout is 3000.
