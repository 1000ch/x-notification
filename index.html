x-notification

Browser Notifications as Web Components.

Example1

<x-notification id='js-noti1' autoshow title='Hello!' delay='0' timeout='3000'>
  This will be shown automatically.
  Delay is 0, timeout is 3000.
</x-notification>

Example2

<x-notification id='js-noti2' title='Notification' delay='500'>
  Delay is 500, no timeout.
  Please click to close this.
</x-notification>

This will be shown automatically. Delay is 0, timeout is 3000. Delay is 500, no timeout. Please click to close this.
